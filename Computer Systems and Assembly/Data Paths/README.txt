Lab 2: Simple Data Path

-----------
DESCRIPTION

In this lab, the user enters two numbers that follow a series of multiplexors, d flip-flops, and ALUs. The program output will result in the first four bit number to be rotated leftward by the amount of the second number.

-----------
FILES

-
Lab2.lgi

This file contains the entire data path.

-----------
INSTRUCTIONS

The program should be run using the application called Multimedia Logic. To run the program, click the play button located on the top of the screen. Enter numbers using the keypad and manipulate the switches by clicking on the switches to turn them on or off.
