Lab 4: Functions and Graphics

DESCRIPTION

In this lab, a series of subroutines and macros are used to create a bitmap display.

FILES

lab4.asm

INSTRUCTIONS

The MARS application should be used to run the program. To start the software, click the wrench icon for the lab4.asm file and the test file. Then, connect the bitmap display by selecting the option 0xffff0000 for the base address and click connect to MIPS. Finally, to run the program, press play.
