Lab 3: ASCII-risks (Asterisks)

DESCRIPTION

In this lab, the user is prompted to enter the height they would like the pattern to be. The program then prints a pattern of stars given the height input by the user.

INSTRUCTIONS

The MARS application should be used to run the program. To start the software, press the play button after clicking the wrench icon. When the application prompts you, enter the height you want the pattern to be and watch it run.
